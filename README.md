# MJUN GAME
A fast-paced 2D Top-down Shooter built in Java using the Swing library.

## 🕹️ Game Features
- **Dynamic Wave System:** Levels 1-3 or **Endless Mode** (Wave 4+).
- **3 Unique Enemy Types:** Thieves, Bandits, and Archers.
- **Weapon System:** Fireball (AOE), Frost Bolt (Freeze/DOT), and Wind Shield (Reflect).
- **Settings & UI:** Fullscreen support, Volume control, and HUD.

## ⌨️ Controls
- **WASD / Arrow Keys:** Movement
- **1, 2, 3:** Switch Weapons
- **Mouse Left Click:** Attack / Use Ability
- **F (in Settings):** Toggle Fullscreen
- **ESC:** Open Settings / Return to Menu

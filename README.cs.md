# MJUN GAME
Akční 2D Top-down střílečka vytvořená v Javě s využitím knihovny Swing.

## 🕹️ Herní Prvky
- **Dynamický systém vln:** Úrovně 1-3 nebo **Endless mód** (Vlna 4+).
- **3 Unikátní typy nepřátel:** Zloději, Bandité a Lukostřelci.
- **Systém zbraní:** Ohnivá koule (Plošné poškození), Mrazivá střela (Zmrazení/DOT) a Větrný štít (Odrážení).
- **Nastavení a UI:** Podpora Fullscreenu, ovládání hlasitosti a HUD.

## ⌨️ Ovládání
- **WASD / Šipky:** Pohyb
- **1, 2, 3:** Přepínání zbraní
- **Levé tlačítko myši:** Útok / Použití schopnosti
- **F (v nastavení):** Zapnout/Vypnout Fullscreen
- **ESC:** Otevřít nastavení / Návrat do menu
